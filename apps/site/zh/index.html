整理媒体库需要的能力，全都内置

从重命名、刮削元数据到从网络下载视频 —— SMM 是你本地媒体库的一站式工具。

智能重命名

一键将文件重命名为 Plex / Jellyfin / Emby 命名规范，自动识别 季、集、标题、年份与语言。

元数据与海报刮削

从 TMDB 与 TVDB 抓取海报、Fanart 与 NFO 文件，让你的媒体库 在任意播放器中都精致美观。

视频下载

基于 yt-dlp，从 YouTube、Bilibili 等网站直接下载视频到媒体库。

视频转码

内置 FFmpeg 转码与封装能力，支持各种主流编码，速度与稳定性兼得。

标签与元数据编辑

直接在应用中编辑标题、剧集、简介等信息，结果以标准 NFO 文件形式 写回到本地。

内置 AI 助手

让 AI 帮你识别媒体目录、修正剧集匹配、生成简介，并通过 MCP Server 供其他工具调用。

自动字幕

通过内置的 VideoCaptioner 工作流，对音频进行转写并生成字幕文件。

跨平台桌面应用

基于 Electron，支持 Windows（含 Windows 11 ARM）、macOS 与 Linux，文件始终保留在你的电脑上。

开始整理你的媒体库

下载对应平台的最新版本，几分钟就能拥有一个井井有条的媒体目录。
