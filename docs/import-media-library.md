# Import Media Library

What is Media Library?
In SMM, the term "media folder" is folder that contains files for TV show, movie or music.
the term "media library" is folder that contains multiple media folders.

For example:
```
/path/to/TvShow # media library
 |- TvShow 1 # media folder 1
 |- TvShow 2 # media folder 2
 |- TvShow 3 # media folder 3

```


This feature allow user to import all media folders at once.
